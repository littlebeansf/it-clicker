## What is IT Clicker?

An infinite idle clicker browser game inspired by Cookie Clicker, but themed around the beautiful disaster that is IT work. 

Click the bug. Earn bugs. Hire overworked interns. Buy server racks that are definitely not on fire. Ascend to godhood via The Mainframe.

## Features

- 🖱️ **Click to earn Bugs** — the only currency that matters
- 🏢 **10 IT-themed buildings** from Rubber Duck Debugger to The Mainframe
- ⚡ **50 upgrades** with office meme descriptions
- 🏆 **25 achievements** — "It Works On My Machine" is the first milestone
- 🔄 **Prestige system** — Reboot the System for CPU Cores that multiply production
- 🎲 **Random events** — Production is down! CEO walked by! Intern pushed to main!
- 💾 **Auto-saves to localStorage** — your suffering persists across browser sessions
- 🎨 **Oldschool pixel art** — all assets generated in 16-bit retro style
- 📴 **100% offline** — pure HTML/JS/CSS, no dependencies, no install
